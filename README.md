# Expense Tracker API

Простой REST API для управления личными расходами, построенный на базе **FastAPI**. Проект создан в рамках изучения архитектуры бэкенд-приложений, валидации данных и автоматизированного тестирования.

## Основной функционал

* **Учет расходов**: Добавление записей с автоматической категоризацией (пометка `is_important` для сумм свыше 10,000).
* **Валидация**: Жесткая проверка входящих данных через **Pydantic** (контроль типов, длин строк и числовых диапазонов).
* **Безопасность**: Защищенные эндпоинты (например, очистка базы) с проверкой прав доступа через кастомные заголовки (**Headers**).
* **Статистика**: Расчет общих затрат и количества операций в реальном времени.
* **Персистентность**: Хранение данных в формате JSON с обработкой ошибок чтения/записи.

## Стек технологий

* **Python 3.13+**
* **FastAPI** (Web Framework)
* **Pydantic** (Data Validation)
* **Pytest** (Testing)
